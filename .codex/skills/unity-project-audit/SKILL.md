# Unity Project Audit

## When to use
- New project onboarding.
- After major imports or pipeline changes.

## Workflow
1. Verify build settings include required scenes.
2. Confirm render pipeline and quality settings.
3. Check active input system (New/Old/Both).
4. Validate tags/layers used by scripts.
5. List missing scripts, broken references, or compile errors.

## Outputs
- A short audit report with fix priorities.
