# Week 10 — 안정화와 릴리즈 준비 (추천 3~5시간)

목표: 10분 플레이 안정성 확인, 문서/태그/빌드 정리를 마무리한다.

## 해야 할 일
- 플레이 테스트
  - 10분 연속 플레이를 녹화하며 예외/끊김/크래시를 확인.
  - 심문/판정 UI, 로그/토스트, G 바가 모두 정상 갱신되는지 체크.
- 버그 픽스
  - 치명/중간 정도의 버그를 우선적으로 수정하고 다시 5분 이상 재검증.
  - 콘솔 경고/에러가 남지 않도록 정리.
- 문서/태그
  - 루트 [`README.md`](../../README.md)와 빌드/실행 가이드 최신화.
  - 릴리즈 노트 초안 작성 후 태그 `v0.1-slice` 후보를 정리.
  - 진행 기록(영상 링크, 주요 변경점, 남은 위험)을 짧게 남긴다.

## DoD
- 10분 플레이에서 예외/치명 버그 없음(영상 기록).
- 문서/태그/릴리즈 노트가 정리돼 있다.
