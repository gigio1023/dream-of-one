# Dream of One — Master Plan (Project.md Driven)

이 문서는 **project.md**를 기반으로 전체 구현 계획, 완료 기준, 진행 현황을 한 곳에 유지한다.
질문 금지. 완료 기준이 충족될 때까지 계속 진행한다.

---

## 0) 목표/비목표

### 목표(Release-Ready)
- **로그 기반 사회 시뮬레이션**이 설계대로 작동
- **결정적 코어**: 위반/의심/신고/판정은 규칙 기반
- **LLM은 표현용 1줄**만 담당(실패 시 폴백)
- 플레이 종료까지 **콘솔 에러 0**

### 비목표(명시)
- [ ] 세이브/로드 기능 없음
- [ ] WEL 재생(replay)으로 상태 복원 없음
- [ ] LLM이 판정/상태 전이 결정 금지

완료 기준(필수)
- Play 즉시 화면/조명/UI 정상 표시
- CITY package 기반 월드 시각화 완료(랜드마크 식별 가능, 동선/Zone 경계 가독)
- **WEL → Blackboard → 주입 → 가십/신고 → 경찰 판정** 루프 1회 이상 자연 재현
- 조직별 **절차 이벤트**가 로그로 기록되고, NPC가 이를 근거로 행동
- 한국어 텍스트 정상 표기(깨짐 없음)

## 0-1) 게임 정체성(핵심 기둥)
- [x] Log-First World: 상태 대신 로그 축적으로 세계 이해
- [x] 조직이 사회를 움직인다: 조직 목표 충돌이 사건 생성
- [x] 결정적 코어: 위반/의심/신고/판정은 규칙 기반
- [x] 텍스트-우선 결과: 1~2줄 텍스트 + 아티팩트로 결과 제시

## 1) 월드/조직 정의

### 1-1. 공간 스펙
- [x] 70m × 70m 도시 블록 스케일 반영

### 1-2. 조직 정의(Goal/Procedure/Resource/Artifact)
#### 스튜디오
- [ ] Goal: RC 제출/릴리즈 안정화
- [ ] Procedure: 칸반 갱신 → 패치노트 → 승인 → RC 삽입
- [ ] Resource: 칸반 보드, 라운지, 서버 슬롯
- [ ] Artifact: 칸반 로그, 패치노트, 승인 노트, RC 스트립

#### 지구대
- [ ] Goal: 경범 처리, 재발 방지
- [ ] Procedure: 신고 접수 → 현장 확인 → 증거 수집 → 심문 → 판정
- [ ] Resource: 티켓 발부, 프린터, 캡처 보드
- [ ] Artifact: 위반 티켓, CCTV 캡처, 사건 로그 스트립

#### 편의점
- [ ] Goal: 품절 0, 라벨 갱신, 거래 질서 유지
- [ ] Procedure: 라벨 점검 → 진열/재고 갱신 → 카운터 규칙 준수
- [ ] Resource: 라벨 시스템, 카운터
- [ ] Artifact: 가격/품절 라벨, 거래 메모

#### 공원관리
- [ ] Goal: 좌석/소음/촬영 규범 유지, 민원 최소화
- [ ] Procedure: 현장 경고 → 조치 → 보고서 작성
- [ ] Resource: 게시판, 조치 권한
- [ ] Artifact: 조치 보고, 게시판 공지, 민원 메모

#### 카페/휴게 공간 운영
- [ ] Goal: 주문 질서/좌석 회전/소음 관리
- [ ] Procedure: 주문 → 대기 → 좌석 안내 → 정리
- [ ] Resource: 주문대, 번호표, 좌석표
- [ ] Artifact: 주문 메모, 좌석/정리 로그

#### 배송/물류
- [ ] Goal: 정시 배송, 출입 규정 준수
- [ ] Procedure: 픽업 → 출입 확인 → 수취 서명 → 반출
- [ ] Resource: 배송 카트, 출입 체크리스트
- [ ] Artifact: 배송 라벨, 수령 서명 로그

#### 시설관리(빌딩 관리)
- [ ] Goal: 시설 안전 유지, 장애 최소화
- [ ] Procedure: 정기 점검 → 수리 요청 → 작업 승인 → 완료 보고
- [ ] Resource: 점검 체크리스트, 작업 허가서
- [ ] Artifact: 점검 로그, 수리 티켓

#### 미디어/촬영팀
- [ ] Goal: 촬영 허가 준수, 촬영 구역 안전 유지
- [ ] Procedure: 사전 허가 → 촬영 구역 표시 → 촬영 → 반납
- [ ] Resource: 촬영 장비, 허가서
- [ ] Artifact: 촬영 허가서, 촬영 로그
